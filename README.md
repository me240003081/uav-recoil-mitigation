# Research on Low Recoil Launch Technology of Small UAV

## Overview
This project investigates on methods to mitigate recoil of firearms mounted on unmanned aerial vehicles (SUAVs) and the principle behind these methods.

## Objectives
- Reduce recoil generated during firing.
- Improve UAV stability and firing accuracy.
- Analytical analysis of the method.
- Numerical analysis of the method.
- Validate the concept using simulation and firing tests.
